Functions are created with object literals. These have four parts:

    
  
  1. reserved word: function
    2. 
  
  2. the optional function name (used for identification, debugging, calling recursively). If no name, it's an anonymous function.
    3. 
  
  3. zero or more parameters in parentheses, which are initialised by the supplied arguments when the function is invoked
  
  4. the body of the function: statements wrapped in curly braces that execute when the function is invoked.



Scope and Closure

Functions can be defined inside other functions. Inner functions can freely access the parameters and variables it is nested within. 
Outer functions can't access variables declared in the inner ones, though. 
This is called Closure.
